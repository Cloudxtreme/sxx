## Overview

sxx is a tool similar to ansible ad-hoc which uses existing ssh, scp, rsync,
etc. syntax.

## Building

To build sxx on all supported platforms, run:

  `scripts/build.sh`

or else do:

  `mkdir bin && cmake .. && make -j3 VERBOSE=1`
